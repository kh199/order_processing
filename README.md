# Сервис обработки заказов

Стек:

```Python```
```FastAPI```
```PostgreSQL```
```RabbitMQ```
```Docker```

Позволяет обрабатывать заказы и отправлять уведомления с помощью очереди сообщений.<br>
Реализованы 3 очереди: очередь для новых заказов, для изменения статуса заказов, для отправки уведомлений клиентам.

## Установка

Клонировать репозиторий:
```
git clone https://github.com/kh199/order_processing
```
Создать и заполнить файл .env на основе .env.template

Запустить docker-compose:
```
docker-compose up -d
```

Документация доступна по адресу ```http://127.0.0.1:8001/docs```

Перед началом работы с заказами нужно создать пользователя с помощью метода API users/create (ввести id чата в телеграме, в который будут приходить сообщения)

## Методы API

+ **POST**   ```orders/create``` создение заказа
+ **POST**   ```orders/change``` изменение статуса заказа
+ **POST**   ```users/create``` создание пользователя
